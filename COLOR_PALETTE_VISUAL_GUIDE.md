## ğŸ“± Responsive Color Usage

### Mobile Best Practices

- **Touch targets:** Use higher contrast colors for small buttons
- **Readability:** Prefer Gray-900 for body text on mobile (better in sunlight)
- **CTAs:** Use bold colors (Primary Blue, Accent Red) for tap targets
- **Backgrounds:** Lighter backgrounds reduce eye strain on mobile

### Desktop Best Practices

- **Hover states:** Always include hover color variants
- **Focus states:** Use visible focus rings (#0B74FF with 2px outline)
- **Large screens:** Can use more subtle color variations (Gray-500, Gray-400)
- **Multi-column:** Use background colors to distinguish sections
